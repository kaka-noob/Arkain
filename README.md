#### 一、四主神

四主神吸收陨落神明的精华而诞生，后面又在阿里顿的偷袭（马保国音）下都死了，有的死透了有的死一半，现在力量都大不如前

- 暗影之主：巴哈瑞泽尔，后来和布莱恩融合形成了新的人格
- 死亡之主：阿里顿，自诩阿克恩的守望者
- 恶魔之主：辛德里奇，死了后爆炸的余波把其它主神炸残了
- 黑暗之主：拉格斯，曾在第一幕第四章给阿里顿打工
